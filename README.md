# Practico1-Sprint3-Nodejs
Este practico se enfoca en agregar endpoints para ampliar la funcionalidad del servidor.
Para desarrollar el practico 1 deben completar la lista de tareas.

Agregar un endpoint que al realizarle una peticion GET que nos devuelva todos los superheroes

Agregar un endpoint que al realizarle una peticion POST cree e inserte un nuevo superheroe en la base de datos, y nos devuelva el superheroe creado

Agregar un endpoint que al realizarle una peticion PUT actualice un superheroe en la base de datos, y nos devuelva todos el superheroe actualizado

Agregar un endpoint que al realizarle una peticion DELETE borre un superheroe por ID en la base de datos, y nos devuelva el superheroe borrado

Agregar un endpoint que al realizarle una peticion DELETE borre un superheroe por Nombre del superheroe en la base de datos, y nos devuelva el superheroe borrado
